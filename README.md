# Jolokia

The Jolokia Ruby library provides a ruby API to the to the Jolokia agent.

## Installation

Add this line to your application's Gemfile:

    gem 'jolokia'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jolokia
